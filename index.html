Подробнее о моей программе

Сайтостроение с нуля

Узнаешь, из каких элементов состоит веб-страница. Как их стилизовать и располагать относительно друг друга. Научишься работать в редакторах кода Codepen и Visual Studio Code. Сделаешь собственноручно небольшой сайт без адаптации.

Вёрстка HTML, CSS+JS

Узнаешь, какую анимацию можно делать с помощью CSS и JavaScript. Научишься проверять адаптивность веб-страницы. Сделаешь свой сайт адаптивным для основных устройств (планшеты, мобильные). Добавишь ещё секций с анимацией элементов.

Программирование JavaScript

Узнаешь возможности и особенности языка JavaScript. Научишься писать простые программы с основными типами данных. Сделаешь часы и/или калькулятор для своего сайта и простую игру Galaxy.

Графика на веб-страницах

Узнаешь, что в HTML есть два графических объекта: CANVAS и SVG. Разберёшься в их отличиях и основных возможностях. Научишься создавать графические элементы средствами CANVAS и SVG, управлять их поведением на веб-странице. Сделаешь простую игру, например, Покемона или Galaxy.

Сайтостроение в WebFlow

Узнаешь, как создавать яркие интерактивные сайты с помощью визуального редактора кода WebFlow. Научишься работать в приложении WebFlow. Сделаешь сайт в WebFlow по выбранному макету, выгрузишь его в виде архива на диск и сможешь разместить на хостинге.

Сайтостроение в WordPress

Узнаешь назначение и основные возможности CMS WordPress. Научишься создавать сайты в WordPress, размещать их на хостинге и изменять с помощью специальных плагинов. Сделаешь сайт в WordPress из выбранной шаблонной темы, настроишь его структуру и оформление по своему прототипу.
